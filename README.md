# MAVSDK test code for the drone developed in the Bakka Collaborative AI Training programme at Bartın University, using Raspberry Pi 5.

## ⚠️ Warning

The codes have only been tested in a simulation environment. In reality, only the **test_arm.py**, **test0.py**, **test1.py**, and **test2.py** codes have been tested.
While testing the **test2.py** code, the drone lost control for still unknown reasons.
